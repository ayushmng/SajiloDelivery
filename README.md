## Setup & Running:

### Prerequisites

- Node.js >= 14.x
- Expo: ~53.0.20
- Expo CLI installed globally (`npm install -g expo-cli`)

### Installation:

```bash
npm install
```

### Run App:

```bash
npx expo start -c
```

### Testing:

```bash
npm test
```

### Code Modification:

# Add below line in tsconfig.json file, replace utils by your new recognizable folder:

# "@/utils/_": ["app/utils/_"]
